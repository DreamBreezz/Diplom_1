## Использованные технологии
* Java 11
* Maven 3.8
* JUnit 4
* Mockito 3.12
* Jacoco 0.8.12


## Настройка проекта
Не требуется.


## Запуск тестов:

Простой прогон тестов с результатом в консоли:
```bash 
mvn clean test
```

Оценка покрытия с помощью плагина Jacoco:
```bash
mvn verify
```
Для просмотра результата откройте файл:
```bash
target/site/jacoco/index.html
```
